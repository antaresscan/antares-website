v1.3.0

Deep Analysis redesign — 4 new tabs replace the originals

  • Insider Watch replaces Timeline. Top 12 wallets in a colour-coded heatmap; each tile shows the wallet’s 6h delta + action (Accumulating / Holding / Reducing / Selling / Dumping). Dev wallet flagged with a purple ring. Wired to holderActivity.rows with criticalActors fallback and concentration-tile fallback when both are missing. #430
  • Buy/Sell Flow replaces Holder Activity. Pressure-gauge hero with ACCUMULATION / DISTRIBUTION / BALANCED status driven by DexScreener pair.txns + pair.volume across 5min / 1h / 6h / 24h windows. Per-window mini cards. #430
  • Wash Volume replaces Outcome Histogram. 0-100 wash score donut + 4 metrics (reported, real estimate, total trades, avg trade size, buy/sell symmetry). Pivoted off Solscan to DexScreener data so it works on every token with a pair. #430
  • Sniper Map replaces Score Breakdown. Concentration retention bar showing top-10 holding share vs distributed float, plus sniper / bundle flag count. Uses Helius top-N with RugCheck fallback so the tab populates even when Helius is unavailable. #431
  • Removed dead buildOutcomeHistogramTab and stale "Outcome Histogram is not available" user-facing string. Marketing copy across index / features / faq / compare / engine / about / terms / token updated to reference the new tabs.
