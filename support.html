Bug report

Feature request

Email with subject [FEATURE]. Tell us the problem you’re trying to solve, not just the feature. Concrete examples beat abstract wishes.

Verdict feedback

Subject [VERDICT] + the contract address. False negatives feed the rug DB; false positives tune the scoring weights.

Security issue

Don’t email exploit details. Use GitHub Private Vulnerability Reporting. Public disclosure before a fix puts users at risk.

One important caveat. Antares is a heuristic risk-screening tool. A SAFE verdict means none of our 7 detection layers found a critical signal at scan time — it is not a guarantee the token won’t rug. False negatives happen, especially on novel attack patterns. Treat the verdict as a high-quality second opinion, not a verdict of innocence. Full disclaimer in Terms.

Frequently asked

Why doesn’t Antares show anything on my page?

Antares only injects on the 8 trading platforms listed in our manifest: DexScreener, Birdeye, Pump.fun, Photon, Axiom, GMGN, Telemetry, GeckoTerminal. On any other site the extension is dormant by design. If you’re on a supported site and still see nothing, refresh the page once — the content script attaches when the URL contains a Solana contract address.

The verdict says DANGER but the price is going up — is Antares wrong?

No. Antares scores structural risk (concentration, contract surface, LP burn status, holder distribution), not short-term price action. A token can pump while still being a coordinated dump risk. The Full Analysis page breaks down which specific layer triggered the verdict so you can decide for yourself.

Why does the same token sometimes give a different score?

Some inputs are real-time: liquidity, holder distribution, volume, and on-chain transfers all change as the token trades. The verdict reflects the state of the chain at scan time. If a critical wallet sells or LP gets pulled, the score moves accordingly.

Does Antares store my wallet address or trades?

No. Antares never reads your wallet, never reads form fields, never reads cookies. It extracts the public Solana contract address visible in the URL or DOM of supported pages and sends only that to our API. Read the privacy policy for the full data flow.

I’m getting “rate limited” errors. What’s happening?

The free public API enforces a sliding-window rate limit so it stays fast for everyone. If you hit it, wait 60 seconds and try again. If you’re consistently hitting the limit, email us — we can usually figure out what’s going on.

Can I use Antares on Solana tokens that aren’t on a DEX yet?

Partially. Pre-launch tokens (no liquidity yet) have no price/holder data, so several layers of the score will be inconclusive. Antares will still surface what it can read on-chain (mint authority, freeze authority, supply, deployer history) but the overall verdict is most reliable once the token is trading.

The overlay disappeared. How do I bring it back?

The overlay can be hidden via the Stealth toggle in the extension popup. Click the Antares icon in your browser toolbar and turn Stealth off. If that’s not it, refresh the page once.

How do I report a token Antares missed or wrongly flagged?

Email antaresantiscam@gmail.com with subject [VERDICT] and include the contract address plus what you observed on-chain. We use false negatives to tune the rug database and false positives to refine the scoring weights.
