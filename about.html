About the project

WHY THIS EXISTS

Antares started because Solana memecoin trading is a minefield. RugCheck does one thing well, GoPlus does another, DexScreener gives you the price chart, BubbleMaps shows the wallet graph — and you have to alt-tab between four sites every time you see a contract address on Twitter. By the time you’ve cross-referenced everything, the price has moved 30%.

So we built the layer that bundles the signals into one verdict, adds what nobody else does (Insider Watch, Buy/Sell Flow, Wash Volume, Sniper Map, insider clustering), and overlays it on every site you already use.

What Antares is, what it isn’t

Antares is a risk-screening tool. It tells you what 7 independent detection layers find about a Solana token in 3-5 seconds. It is not financial advice, it does not predict the future, and it cannot guarantee a SAFE token won’t rug. False positives and false negatives both happen — we trade off in favour of safety (better one false alarm than one missed honeypot).

The Wash Volume score and Sniper Map retention are indicative — derived from heuristic signals (avg trade size relative to liquidity, top-10 concentration), not validated forecasts. Treat them as context, not certainty. See Terms for the full disclaimer.

How it stays free

No VC

No outside investors, no funding round. The project is funded out-of-pocket and runs on free-tier infrastructure (Vercel hobby + Upstash free + free Helius + free Gemini).

No token

No ANTARES coin, no airdrop, no points. The project will never launch a token. If you see one, it’s a scam impersonating us.

No paywall

The extension is and will remain free. Power-user features (custom alerts, watchlists, public API) may eventually be a tip-funded layer, but the core verdict will always be free.

Open source

The full source is on GitHub. Audit the scoring logic, fork it, self-host it. The license is permissive and the API is documented.
