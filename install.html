Install in 60 seconds

GET ANTARES.

Free Chrome extension. No account, no signup, no data collection. Works on Chrome, Brave, Edge, Arc, and any Chromium-based browser. Activates automatically on supported Solana trading platforms.

01

Open the Chrome Web Store

Antares is published on the Chrome Web Store. Click the button below to open the listing.

Get notified at launch
Listing pending approval. Until the Chrome Web Store review is complete, the link above goes to the GitHub repository where you can sideload the extension manually (load unpacked from the latest build/chrome-mv3-prod). Once the listing is live, this page will update automatically.
02

Click "Add to Chrome"

The Web Store will show a permission dialog. Antares requests minimum permissions:

• storage — remember your settings locally (theme, alert thresholds)
• notifications — alert you when a token’s verdict escalates
• activeTab + tabs — read the URL and visible DOM on supported sites only
• alarms — periodic background re-scans

No password access, no form-fill access, no browsing history. The extension is dormant on every site that isn’t a supported Solana platform. Full breakdown in /privacy.
03

Browse normally

Antares activates automatically when you land on a supported Solana platform:

• DexScreener (Solana pairs)
• pump.fun
• Axiom
• Photon
• Birdeye
• GeckoTerminal
• GMGN.ai
• Telemetry

Every Solana contract address you encounter triggers an automatic scan. The verdict appears as an overlay panel within 2–5 seconds. The Chrome toolbar icon shows a colour-coded badge (✓ / ! / ✕) at a glance.
